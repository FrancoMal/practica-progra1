Modos de apertura

rRead - Solo lectura. Archivo DEBE existir.
wWrite - Solo escritura. Si existe lo DESTRUYE.
aAppend - Escritura al final. No borra.
tText - Modificador: "rt", "wt", "at". Default: "rt".

Rutas en Windows

# MAL - \n es salto de linea:
open("c:\nuevo\datos.txt", "wt")
# BIEN:
open("c:\\nuevo\\datos.txt", "wt")  # doble barra
open("c:/nuevo/datos.txt", "wt")   # barra normal
open(r"c:\nuevo\datos.txt", "wt")  # raw string

Estructura obligatoria

try:
    arch = open("datos.txt", "rt")
    # procesamiento...
except FileNotFoundError as msg:
    print("No encontrado:", msg)
except OSError as msg:
    print("Error:", msg)
finally:
    try:
        arch.close()
    except NameError:
        pass
