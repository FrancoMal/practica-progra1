Concepto de Cadena

  • Una cadena de caracteres (string) es un conjunto de 0 o mas caracteres.
  • Python 3 usa codificacion UTF-8, que soporta caracteres regionales (enie, tildes, etc).
  • Las cadenas son secuencias inmutables de caracteres.

Constantes de cadena

cad1 = "Lunes"           # comillas dobles
cad2 = 'Martes'          # comillas simples
marca = "Mc. Donald's"   # dobles permiten simple adentro
frase = 'Dijo "Ya voy"'  # simples permiten dobles adentro

Cadenas multilinea y continuacion

# Continuacion con barra invertida:

Diferencia clave: INMUTABILIDAD

Las listas son mutables, las cadenas son inmutables. No se pueden modificar directamente.

cad = "Hola"
cad[0] = "h"   # ERROR! TypeError
cad = "h" + cad[1:]  # Se crea una NUEVA cadena: "hola"

Los metodos de cadenas devuelven una nueva cadena, sin alterar la original.

No existe reverse() - Usar [::-1]

original = "Hola"
invertida = original[::-1]  # "aloH"

Comparacion

Se comparan alfabeticamente (como en un diccionario). Mayusculas van antes que minusculas.

"Ana" < "Beto"    # True
"ana" > "Ana"     # True (minusc > mayusc en UTF-8)

Conversion: str(), int(), float()

c = 75
msg = "Precio: " + str(c) + " dolares"  # concatenar numero

dato = "25"
n = int(dato)       # 25 (entero)

valor = "3.1416"
pi = float(valor)   # 3.1416 (real)

Metodos - Clase 5

split([sep])

Divide la cadena en una lista. Si se omite sep, separa por espacios (multiples espacios = uno).

frase = "Hola Mundo"
lista = frase.split()        # ['Hola', 'Mundo']

fecha = "21/9/2020"
d, m, a = fecha.split('/')  # d='21', m='9', a='2020'

# Diferencia importante:
cad = "Hola   Mundo"   # 3 espacios
cad.split()          # ['Hola', 'Mundo']           (sin param)
cad.split(' ')       # ['Hola', '', '', 'Mundo']   (con ' ')

sep.join(secuencia)

Une los elementos de una secuencia con el separador.

dias = ["lunes", "martes", "jueves"]
cad = ' - '.join(dias)
print(cad)  # lunes - martes - jueves

replace(viejo, nuevo [, max])

Devuelve una cadena nueva con reemplazos. No da error si no encuentra.

a = "Hoy es un dia frio, que frio esta!"
b = a.replace("frio", "humedo")
# "Hoy es un dia humedo, que humedo esta!"

c = a.replace("frio", "humedo", 1)
# "Hoy es un dia humedo, que frio esta!"  (solo 1 reemplazo)

Metodos - Clase 6

Metodos is... (devuelven True/False)

MetodoDescripcionEjemplo
isalpha()True si todos son letras"Hola".isalpha() → True
isdigit()True si todos son digitos"1234".isdigit() → True
"3.14".isdigit() → False
isalnum()True si todos son letras o digitos"XR150".isalnum() → True
"-1234".isalnum() → False

isupper() e islower() ignoran caracteres no alfabeticos.

upper(), lower(), capitalize(), title()

'hola 123'.upper()         # 'HOLA 123'
'HOLA 123'.lower()         # 'hola 123'
'lunes MARTES'.capitalize() # 'Lunes martes' (solo 1ra letra mayusc)
'lunes MARTES'.title()      # 'Lunes Martes' (1ra de cada palabra)

center(), ljust(), rjust(), zfill()

'Hola'.center(10, '-')   # '---Hola---'
'Hola'.ljust(10, '-')    # 'Hola------'
'Hola'.rjust(10, '-')    # '------Hola'
str(3).zfill(5)            # '00003'

strip(), lstrip(), rstrip()

Eliminan caracteres de los extremos. Sin parametro, eliminan espacios.

'--> Salida -->'.lstrip('->')   # ' Salida -->'
'Continuara...'.rstrip('.')    # 'Continuara'
'  Hola  '.strip()              # 'Hola'

lista = [1, 2, 3, 4]
cad1 = str(lista)                 # '[1, 2, 3, 4]'
cad2 = cad1.strip("[ ]")          # '1, 2, 3, 4'

strip elimina CUALQUIERA de los caracteres dados, no la subcadena completa.

find() y rfind()

find() busca la primera aparicion. rfind() busca la ultima. Devuelven -1 si no encuentran (a diferencia de index() que da error).

cad = "Hola Mundo Hola"
cad.find("Hola")      # 0  (primera aparicion)
cad.rfind("Hola")     # 11 (ultima aparicion)
cad.find("xyz")       # -1 (no encontrado, sin error)

startswith() y endswith()

cad = "archivo.txt"
cad.startswith("arch")  # True
cad.endswith(".txt")     # True
cad.endswith(".pdf")     # False

f-strings (Cadenas formateadas)

Sintaxis basica

dia = 16
mes = "Enero"
cad = f"Sucedio el {dia} de {mes}."
# "Sucedio el 16 de Enero."

# Expresiones y metodos dentro de {}:
print(f"{2 * 37}")              # 74
print(f"{'Lunes'.lower()}")     # lunes

Alineacion y formato

num = 15
pal = "Hola"

# Ancho:
f"*{num:5}*"       # '*   15*'  (numeros a la derecha)
f"*{pal:8}*"       # '*Hola    *'  (strings a la izquierda)

# Relleno con ceros:
f"*{num:05}*"      # '*00015*'

# Alineacion explicita: < izquierda, > derecha, ^ centro
f"*{num:<8}*"      # '*15      *'
f"*{pal:>8}*"      # '*    Hola*'
f"*{pal:^8}*"      # '*  Hola  *'

# Caracter de relleno personalizado:
f"*{pal:=^8}*"     # '*==Hola==*'

# Decimales:
pi = 3.1416
f"{pi:.2f}"        # '3.14'

Copia de cadenas

Copiar una cadena siempre copia su referencia (direccion de memoria). Todas apuntan al mismo objeto porque son inmutables.

a = "Hola"
b = a
c = a[:]
d = str(a)
# Todas tienen el MISMO id() - misma direccion de memoria
