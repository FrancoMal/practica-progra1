Teoria Completa - Manejo de Archivos

Conceptos fundamentales

  • Archivo = conjunto de registros
  • Registro = conjunto de campos
  • Los registros se separan con \n (salto de linea)
  • En archivos CSV, los campos se separan con ; (punto y coma)
# Ejemplo de archivo "alumnos.txt":
# Juan;8;9;7\n    ← registro 1 (campos: nombre, nota1, nota2, nota3)
# Maria;10;6;8\n   ← registro 2
# Pedro;5;4;6\n   ← registro 3

Modos de apertura

  • "r" - Lectura (read). El archivo DEBE existir. Es el modo por defecto.
  • "w" - Escritura (write). DESTRUYE el contenido existente. Crea si no existe.
  • "a" - Agregar (append). Escribe al final del archivo. Crea si no existe.
  • "t" - Texto (modificador). Es el modo por defecto.
  • El modo por defecto es "rt" (lectura + texto).
arch = open("datos.txt", "r")   # lectura
arch = open("datos.txt", "w")   # escritura (BORRA todo!)
arch = open("datos.txt", "a")   # agregar al final
arch = open("datos.txt")        # equivale a "rt"

Rutas en Windows: el problema del \n

Windows usa \ en las rutas, pero \n es salto de linea!

# MAL - \n se interpreta como salto de linea
arch = open("C:\nueva_carpeta\notas.txt")

# SOLUCIONES:
arch = open("C:\\nueva_carpeta\\notas.txt")  # doble barra
arch = open("C:/nueva_carpeta/notas.txt")   # barra normal
arch = open(r"C:\nueva_carpeta\notas.txt")  # raw string

Estructura OBLIGATORIA: try/except/finally

SIEMPRE usar try/except/finally

NUNCA abrir un archivo sin try/except/finally.

try:
    arch = open("datos.txt", "r")
    # ... procesamiento ...
except FileNotFoundError:
    print("Error: archivo no encontrado")
except OSError:
    print("Error de sistema")
finally:
    try:
        arch.close()
    except NameError:
        pass
  • FileNotFoundError ANTES que OSError (es mas especifico)
  • close() en finally para que se cierre SIEMPRE (incluso si hay error)
  • try/except NameError en el close porque si open() fallo, arch no existe

Metodos de lectura: PERMITIDO vs PROHIBIDO

PERMITIDO readline() - METODO PREFERIDO

Lee UNA linea a la vez. Retorna "" (string vacio) al llegar al final.

try:
    arch = open("datos.txt", "r")
    linea = arch.readline()
    while linea != "":
        linea = linea.rstrip("\n")  # quitar el \n del final
        print(linea)
        linea = arch.readline()
except FileNotFoundError:
    print("Archivo no encontrado")
except OSError:
    print("Error de sistema")
finally:
    try:
        arch.close()
    except NameError:
        pass

PERMITIDO for linea in arch

Recorre linea por linea automaticamente.

try:
    arch = open("datos.txt", "r")
    for linea in arch:
        linea = linea.rstrip("\n")
        print(linea)
except FileNotFoundError:
    print("Archivo no encontrado")
except OSError:
    print("Error de sistema")
finally:
    try:
        arch.close()
    except NameError:
        pass

ACEPTABLE read(1) - leer de a un caracter

car = arch.read(1)
while car != "":
    print(car, end="")
    car = arch.read(1)

PROHIBIDO readlines() - carga TODO en lista

lineas = arch.readlines()  # PROHIBIDO! Carga todo a memoria
for linea in lineas:
    print(linea)

PROHIBIDO read() sin parametro - carga TODO en string

contenido = arch.read()  # PROHIBIDO! Carga todo a memoria
print(contenido)

PROHIBIDO list(arch) - carga TODO en lista

lineas = list(arch)  # PROHIBIDO! Carga todo a memoria

REGLA DE ORO NUNCA cargar el archivo completo en memoria

"Otros lo hacen. Nosotros NO."

No importa si es en un string, en una lista, o en cualquier estructura. SIEMPRE procesar linea por linea.

Escritura

write() y writelines()

# Escribir lineas - agregar \n manualmente
arch = open("salida.txt", "w")
arch.write("Juan;8;9;7\n")    # UNA write por registro completo
arch.write("Maria;10;6;8\n")

# writelines() para lista de strings
lineas = ["linea1\n", "linea2\n"]
arch.writelines(lineas)

seek(0) - Volver al inicio

BIEN Doble pasada con seek(0)

Si necesitas leer el archivo dos veces, usa seek(0). NO cierres y vuelvas a abrir.

# Primera pasada: contar registros
cont = 0
for linea in arch:
    cont += 1

arch.seek(0)  # volver al inicio

# Segunda pasada: procesar
for linea in arch:
    print(linea.rstrip("\n"))

MAL Cerrar y reabrir para releer

arch.close()
arch = open("datos.txt", "r")  # NO! Usar seek(0)

PROHIBIDO Mezclar for + readline()

for linea in arch:
    otra = arch.readline()  # PIERDE DATOS!

Mezclar for con readline() en el mismo archivo causa perdida de datos.

Reglas adicionales importantes

NO abrir y cerrar por cada registro

# MAL - abre y cierra el archivo por cada registro
for i in range(10):
    arch = open("datos.txt", "a")
    arch.write(dato + "\n")
    arch.close()

# BIEN - abrir una vez, escribir todo, cerrar al final
arch = open("datos.txt", "a")
for i in range(10):
    arch.write(dato + "\n")
arch.close()

NO escribir campo por campo

# MAL - multiples write por registro
arch.write(nombre)
arch.write(";")
arch.write(str(nota))
arch.write("\n")

# BIEN - un solo write por registro completo
arch.write(nombre + ";" + str(nota) + "\n")

Otras reglas

  • Minimizar pasadas por el archivo (no leer 5 veces si se puede hacer en 1 o 2)
  • encoding="UTF8" solo cuando sea estrictamente necesario
  • NO pedirle al usuario el nombre del archivo
  • Formato CSV: campos separados por ; (punto y coma)

Ejemplo completo: Leer CSV y calcular promedio

# Archivo alumnos.csv: nombre;nota1;nota2;nota3
try:
    arch = open("alumnos.csv", "r")
    linea = arch.readline()
    while linea != "":
        linea = linea.rstrip("\n")
        campos = linea.split(";")
        nombre = campos[0]
        n1 = int(campos[1])
        n2 = int(campos[2])
        n3 = int(campos[3])
        prom = (n1 + n2 + n3) / 3
        print(nombre, "-> Promedio:", round(prom, 2))
        linea = arch.readline()
except FileNotFoundError:
    print("Archivo no encontrado")
except OSError:
    print("Error de sistema")
finally:
    try:
        arch.close()
    except NameError:
        pass

Resumen de Reglas Clave

1. SIEMPRE try/except/finally con close en finally

2. NUNCA cargar archivo completo en memoria (read(), readlines(), list())

3. NUNCA mezclar for con readline()

4. NUNCA abrir/cerrar por cada registro

5. UN solo write por registro completo (no campo por campo)

6. Usar seek(0) para releer, NO cerrar y reabrir

7. FileNotFoundError ANTES que OSError

8. NO pedir al usuario el nombre del archivo
