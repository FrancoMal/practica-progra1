Conceptos Fundamentales

Que es un archivo?

Un archivo es un conjunto de elementos llamados registros, todos del mismo tipo de dato, almacenado en un dispositivo auxiliar para preservar la informacion.

Estructura

ARCHIVO
Conjunto de registros
➔
REGISTRO
Conjunto de campos
➔
CAMPO
Un dato individual
1042735;Vignale, Juan Jose       ← registro (2 campos: legajo y nombre)
1118693;Garay, Mariela Daiana    ← otro registro
1094219;Zanini, Candela Belen    ← otro registro

Las 3 etapas

1 APERTURA
open()
➔
2 PROCESAMIENTO
read / write
➔
3 CIERRE
close()

Delimitadores

Cada registro se separa con \n. El caracter real depende del SO:

  • Windows: CR + LF
  • Linux/Android: LF
  • macOS: CR

Apertura de Archivos

Sintaxis

variable = open("nombre_archivo", "modo")

Modos de apertura

rRead - Solo lectura. El archivo DEBE existir.
wWrite - Solo escritura. Si existe, lo DESTRUYE. Si no, lo crea.
aAppend - Solo escritura al final. Si no existe, lo crea.
tText - Modificador: "rt", "wt", "at"

Si se omite el modo, se asume "rt".

Rutas en Windows

"c:\nuevo\datos.txt" es INVALIDA (\n = salto de linea). Soluciones:

# 1. Doble barra
open("c:\\nuevo\\datos.txt", "wt")
# 2. Barra normal
open("c:/nuevo/datos.txt", "wt")
# 3. Cadena cruda
open(r"c:\nuevo\datos.txt", "wt")

Estructura completa con try/except/finally

try:
    arch = open("datos.txt", "rt")
    # procesamiento...
except FileNotFoundError as msg:
    print("Archivo no encontrado:", msg)
except OSError as msg:
    print("Error de archivo:", msg)
finally:
    try:
        arch.close()
    except NameError:
        pass

FileNotFoundError siempre ANTES de OSError (es mas especifica).

Metodos de Lectura

PERMITIDO readline()

Lee UNA linea. Devuelve "" al final.

linea = arch.readline()
while linea:
    # procesar
    linea = arch.readline()

PROHIBIDO readlines()

Carga TODO en una lista.

lineas = arch.readlines()
# PROHIBIDO en el curso!

PERMITIDO for linea in arch

for linea in arch:
    # una linea por vez

PROHIBIDO read() sin param

contenido = arch.read()
# Todo en un string!

NUNCA cargar archivos completos en memoria

Memoria = string, lista, o cualquier estructura. Otros lo hacen. Nosotros NO.

seek(0) para releer

arch.seek(0)  # Vuelve al inicio sin cerrar

Solo cerrar y reabrir si cambia el modo de apertura.

Metodos de Escritura

write() - agrega \n manualmente

arch.write(lu + ';' + nombre + '\n')

writelines() - lista de cadenas

lista = ["Linea1\n", "Linea2\n"]
arch.writelines(lista)

Ejemplo: Crear archivo CSV

try:
    arch = open("alumnos.txt", "wt")
    lu = input("LU? (Enter para terminar): ")
    while lu != "":
        nombre = input("Nombre? ")
        arch.write(lu + ';' + nombre + '\n')
        lu = input("LU? (Enter para terminar): ")
    print("Archivo creado correctamente.")
except OSError as msg:
    print("No se puede grabar:", msg)
finally:
    try:
        arch.close()
    except NameError:
        pass

Ejemplo: Copiar archivo convirtiendo a mayusculas

try:
    entrada = open("notas.txt", "rt")
    salida = open("notas2.txt", "wt")
    k = 0
    for linea in entrada:
        salida.write(linea.upper())
        k += 1
except FileNotFoundError as msg:
    print("No se puede abrir:", msg)
except OSError as msg:
    print("Error:", msg)
else:
    print("Lineas copiadas:", k)
finally:
    try:
        entrada.close()
        salida.close()
    except NameError:
        pass

Los 7 Errores que te Desaprueban

1. Cargar todo en memoria

MAL
contenido = arch.read()
lineas = arch.readlines()
lista = list(arch)
BIEN
for linea in arch:
    # una por vez
linea = arch.readline()

2. Mezclar for con readline

for linea in arch:
    print(linea)
    linea = arch.readline()  # SALTA REGISTROS!

3. Abrir y cerrar por cada registro

for i in range(n):
    arch = open("datos.txt", "at")
    arch.write(dato + '\n')
    arch.close()

4. No usar try/except/finally

arch = open("datos.txt", "rt")
for linea in arch:
    print(linea)
arch.close()  # Si falla open, crash!

5. No sacar el \n al leer

print(linea)  # Doble salto de linea!
# Correcto:
print(linea.rstrip('\n'))

6. Cerrar y reabrir para releer (mismo modo)

arch.close()
arch = open("datos.txt", "rt")  # Costoso!
# Correcto:
arch.seek(0)

7. Leer/grabar por campos separados

arch.write(lu)
arch.write(';')
arch.write(nombre)
arch.write('\n')
# Correcto: un solo write
arch.write(lu + ';' + nombre + '\n')

Ejercicio 1: Imprimir lineas con la letra "a" Facil

Ejercicio 3: Copiar registros filtrados Medio

Ejercicio 4: Encontrar el mayor precio Medio

"ventas.txt" tiene formato producto;cantidad;precio_unitario. Encuentra el producto con mayor precio unitario e imprimilo.

Ejercicio 5: Archivo de entrada y salida con calculo Medio

"empleados.txt" tiene nombre;sueldo_basico;antiguedad. Crea "sueldos.txt" con nombre;sueldo_final donde sueldo_final = sueldo_basico * (1 + antiguedad * 0.03).

Ejercicio 6: Contar palabras totales Facil

Lee "texto.txt" y cuenta la cantidad total de palabras en todo el archivo.

Ejercicio 7: Buscar un registro especifico Medio

"alumnos.txt" tiene legajo;nombre. Dado un legajo ingresado por teclado, buscar si existe e imprimir el nombre. Si no existe, informar.

Ejercicio 8: Agregar registros (append) Medio

Agrega nuevos alumnos al final de "alumnos.txt" sin perder los datos anteriores. Pedir legajo y nombre hasta que se ingrese legajo vacio.

Ejercicio 9: Contar registros y acumular Dificil

"ventas.txt" tiene producto;cantidad;precio_unitario. Calcula e imprime: a) cantidad total de registros, b) monto total vendido (cantidad * precio de cada registro), c) promedio de monto por venta.

Ejercicio 10: Recorrer dos veces con seek Dificil

"numeros.txt" tiene un numero por linea. En el PRIMER recorrido calcula el promedio. En el SEGUNDO recorrido (usando seek) imprime solo los numeros mayores al promedio. Todo en una sola apertura.
