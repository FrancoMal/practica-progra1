Modos de apertura

rRead - Solo lectura. Archivo DEBE existir.
wWrite - Solo escritura. Si existe lo DESTRUYE.
aAppend - Escritura al final. No borra.
tText - Modificador: "rt", "wt", "at". Default: "rt".

Rutas en Windows

# MAL - \n es salto de linea:
open("c:\nuevo\datos.txt", "wt")
# BIEN:
open("c:\\nuevo\\datos.txt", "wt")  # doble barra
open("c:/nuevo/datos.txt", "wt")   # barra normal
open(r"c:\nuevo\datos.txt", "wt")  # raw string

Estructura completa obligatoria

try:
    arch = open("datos.txt", "rt")
    # procesamiento...
except FileNotFoundError as msg:
    print("No encontrado:", msg)
except OSError as msg:
    print("Error:", msg)
finally:
    try:
        arch.close()
    except NameError:
        pass

FileNotFoundError ANTES de OSError. finally con try/except NameError.

Lectura

OK readline()

linea = arch.readline()
while linea:
    # procesar
    linea = arch.readline()

PROHIBIDO readlines()

lineas = arch.readlines()

Carga TODO en lista.

OK for linea in arch

for linea in arch:
    # una linea por vez

PROHIBIDO read()

todo = arch.read()

Todo en un string.

NUNCA cargar archivo completo en memoria

Otros lo hacen. Nosotros NO. Memoria = string, lista, cualquier estructura.

seek(0) para releer

arch.seek(0)  # vuelve al inicio

Solo cerrar/reabrir si cambia el modo.

Escritura

write() - \n manual

arch.write(lu + ';' + nombre + '\n')  # un write por registro

Los 7 Errores que Desaprueban

1. Cargar todo en memoria

arch.read() / arch.readlines() / list(arch)

2. Mezclar for con readline

for linea in arch:
    x = arch.readline()  # pierde datos

3. Abrir/cerrar por cada registro

for i in range(n):
    arch = open(...); arch.write(...); arch.close()

4. Sin try/except/finally

5. No sacar \n al leer

print(linea)  # doble salto
print(linea.rstrip('\n'))  # correcto

6. Cerrar/reabrir mismo modo en vez de seek(0)

7. Grabar campo por campo (varios write por registro)
