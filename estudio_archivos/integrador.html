1. Teoria - Resumen General

Funciones

  • def nombre_funcion(parametros): ... return valor
  • Variables locales: solo existen dentro de la funcion
  • Parametros inmutables (int, str, tuple): NO se modifican afuera
  • Parametros mutables (list, dict): SI se modifican afuera
  • lambda: funciones anonimas de una expresion: lambda x: x * 2
  • Modulos: import modulo / from modulo import funcion
def calcular_promedio(lista):
    # UNA sola tarea, NO input, NO print
    if len(lista) == 0:
        return 0
    return sum(lista) / len(lista)

PERMITIDO

  • Una tarea por funcion
  • return al final de la funcion
  • Funciones definidas al inicio
  • Separar: input en main, calculo en funcion
  • Documentar con docstrings

PROHIBIDO

  • return dentro de un loop (for/while)
  • input() dentro de funciones de calculo
  • print() dentro de funciones de calculo
  • Funcion que haga input + calculo + print
  • Parametro mutable por defecto: def f(lista=[])

Listas

  • Creacion: lista = [1, 2, 3] o lista = list()
  • Indexing: lista[0], lista[-1]
  • Slicing: lista[1:3], lista[::-1]
  • Metodos: append, insert, pop, remove, sort, reverse, index, count
  • Copia: copia = lista[:] o copia = list(lista)
  • List comprehension: [x**2 for x in lista if x > 0]
  • map/filter: list(map(int, lista_str))
# Copia vs referencia
a = [1, 2, 3]
b = a          # b apunta a la MISMA lista (referencia)
c = a[:]       # c es una COPIA independiente
d = list(a)    # d es otra COPIA independiente

PERMITIDO

  • append() para agregar al final
  • insert(pos, elem) en posicion
  • sort() / sorted() para ordenar
  • Copiar con [:] o list()
  • List comprehension

PROHIBIDO

  • Modificar lista mientras se itera con for
  • Asignar lista como parametro default: def f(l=[])
  • Confundir referencia con copia
  • Usar remove() sin verificar existencia
  • Acceder indice fuera de rango sin control

Cadenas (Strings)

  • Inmutables: no se pueden cambiar caracteres individuales
  • Indexing/slicing igual que listas: s[0], s[1:4]
  • split(sep): separa en lista. "a;b;c".split(';') da ['a','b','c']
  • join(): une lista en string. ';'.join(lista)
  • replace(old, new): reemplaza subcadenas
  • upper()/lower(): mayusculas/minusculas
  • strip()/rstrip()/lstrip(): quita espacios o caracteres
  • find(sub): devuelve posicion o -1
  • f-strings: f"Hola {nombre}, nota: {nota:.2f}"
  • Escape: \n salto, \t tab, \\ barra

PERMITIDO

  • split() para separar campos CSV
  • rstrip('\n') para limpiar lineas
  • f-strings para formatear salida
  • Comparar con == (case-sensitive)
  • in para buscar subcadena

PROHIBIDO

  • Intentar modificar s[0] = 'X' (inmutable!)
  • Olvidar que split devuelve STRINGS (hay que convertir)
  • Olvidar rstrip('\n') al leer de archivo
  • Concatenar con + en loops (ineficiente)

Excepciones

  • try/except/else/finally: estructura de manejo de errores
  • Orden: excepciones especificas ANTES de generales
  • FileNotFoundError antes de OSError (es mas especifica)
  • raise: lanzar excepcion manualmente
  • assert: verificar condicion (debug)
  • finally: SIEMPRE se ejecuta (ideal para cerrar archivos)
try:
    resultado = x / y
except ZeroDivisionError:
    print("No se puede dividir por cero")
except TypeError:
    print("Tipo de dato incorrecto")
else:
    print("Todo salio bien")
finally:
    print("Siempre se ejecuta")

PERMITIDO

  • FileNotFoundError ANTES de OSError
  • finally para cerrar archivos
  • try/except NameError en finally (proteger close)
  • ValueError para conversion de tipos
  • else para codigo que solo corre si no hubo error

PROHIBIDO

  • except sin tipo: except: (atrapa todo)
  • OSError antes de FileNotFoundError
  • Ignorar excepciones con pass sin justificar
  • Usar excepciones para control de flujo normal

Archivos

  • 3 etapas: open() ➔ proceso ➔ close()
  • Modos: "r" lectura, "w" escritura (destruye!), "a" append
  • Lectura: readline() o for linea in arch
  • Escritura: write(dato + '\n') - un write por registro
  • seek(0) para volver al inicio sin cerrar
  • CSV: campos separados por ; , usar split(';')
# Estructura OBLIGATORIA para archivos
try:
    arch = open("datos.txt", "rt")
    for linea in arch:
        linea = linea.rstrip('\n')
        # procesar...
except FileNotFoundError as msg:
    print("No encontrado:", msg)
except OSError as msg:
    print("Error:", msg)
finally:
    try:
        arch.close()
    except NameError:
        pass

PERMITIDO

  • readline() con while
  • for linea in arch:
  • write() con \n manual
  • seek(0) para releer
  • try/except/finally SIEMPRE
  • rstrip('\n') al leer

PROHIBIDO

  • readlines() - carga todo en lista
  • read() - carga todo en string
  • list(arch) - carga todo en lista
  • Mezclar for con readline()
  • Abrir/cerrar por cada registro
  • Cerrar/reabrir mismo modo (usar seek(0))
  • Varios write() por registro

Recursividad

  • Caso base: condicion de parada (sin llamada recursiva)
  • Caso recursivo: llama a si misma con problema mas chico
  • Toda recursion puede hacerse iterativa (y viceversa)
  • Ventaja: codigo mas elegante y natural para ciertos problemas
  • Desventaja: usa mas memoria (pila de llamadas), puede ser mas lenta
def factorial(n):
    if n <= 1:          # caso base
        return 1
    return n * factorial(n - 1)  # caso recursivo

def fibonacci(n):
    if n <= 0: return 0
    if n == 1: return 1
    return fibonacci(n-1) + fibonacci(n-2)

def potencia(base, exp):
    if exp == 0: return 1
    return base * potencia(base, exp - 1)

PERMITIDO

  • Caso base con if simple (sin loops)
  • Reducir el problema en cada llamada
  • return con llamada recursiva
  • Multiples casos base si es necesario

PROHIBIDO

  • for/while para determinar caso base
  • Recursion sin caso base (loop infinito)
  • No reducir el problema (loop infinito)
  • Variables globales en recursion

Problema 1: Conteo de Votos

Consigna: Se tiene un archivo "votos.txt" donde cada linea tiene el formato partido;lista. Se pide:

  1. Leer el archivo y contar la cantidad de votos por partido (usar diccionario)
  2. Mostrar el partido ganador (el que mas votos tiene)
  3. Mostrar un grafico de barras con asteriscos (1 asterisco = 1 voto)

Contenido de prueba para votos.txt:

Partido_A;101
Partido_B;202
Partido_A;101
Partido_C;303
Partido_B;202
Partido_A;101
Partido_A;102
Partido_B;201
Partido_C;303
Partido_A;101
Pistas: Temas involucrados: archivos + diccionarios + funciones + cadenas. Usar dict.get(clave, 0) para contar. Usar max() con key para encontrar ganador.
Ver solucion 
def contar_votos(nombre_archivo):
    votos = {}
    try:
        arch = open(nombre_archivo, "rt")
        for linea in arch:
            linea = linea.rstrip('\n')
            partido, lista = linea.split(';')
            votos[partido] = votos.get(partido, 0) + 1
    except FileNotFoundError as msg:
        print("No encontrado:", msg)
    except OSError as msg:
        print("Error:", msg)
    finally:
        try:
            arch.close()
        except NameError:
            pass
    return votos

def mostrar_ganador(votos):
    if len(votos) == 0:
        print("No hay votos.")
        return
    ganador = max(votos, key=lambda p: votos[p])
    print(f"Ganador: {ganador} con {votos[ganador]} votos")

def grafico_barras(votos):
    for partido, cant in votos.items():
        print(f"{partido:15s} | {'*' * cant} ({cant})")

# Programa principal
votos = contar_votos("votos.txt")
if votos:
    mostrar_ganador(votos)
    grafico_barras(votos)

Problema 2: Gestion de Alquiler de Items

Consigna: Sistema de alquiler con menu. Usar un diccionario donde la clave es el codigo del item y el valor es el nombre del cliente.

  1. Registrar alquiler: pedir codigo y cliente, agregar al dict. Si ya esta alquilado, avisar.
  2. Devolver item: pedir codigo, eliminar del dict. Si no esta alquilado, avisar.
  3. Reporte: mostrar todos los items alquilados ordenados por codigo.
  4. Salir

El menu se repite hasta que el usuario elija salir. Usar excepciones para entrada invalida.

Pistas: Temas: diccionarios + funciones + excepciones + while + listas (sorted). Cada opcion del menu deberia ser una funcion separada.
Ver solucion 
def registrar(alquilados):
    codigo = input("Codigo del item: ")
    if codigo in alquilados:
        print(f"Item {codigo} ya alquilado por {alquilados[codigo]}")
    else:
        cliente = input("Nombre del cliente: ")
        alquilados[codigo] = cliente
        print("Alquiler registrado.")

def devolver(alquilados):
    codigo = input("Codigo del item a devolver: ")
    if codigo in alquilados:
        del alquilados[codigo]
        print("Devolucion registrada.")
    else:
        print("Ese item no esta alquilado.")

def reporte(alquilados):
    if len(alquilados) == 0:
        print("No hay items alquilados.")
        return
    print("--- Items Alquilados ---")
    for codigo in sorted(alquilados):
        print(f"  {codigo} -> {alquilados[codigo]}")

# Programa principal
alquilados = {}
opcion = ""
while opcion != "4":
    print("\n1. Registrar alquiler")
    print("2. Devolver item")
    print("3. Reporte")
    print("4. Salir")
    opcion = input("Opcion: ")
    if opcion == "1":
        registrar(alquilados)
    elif opcion == "2":
        devolver(alquilados)
    elif opcion == "3":
        reporte(alquilados)
    elif opcion == "4":
        print("Fin.")
    else:
        print("Opcion invalida.")

Problema 3: Asignacion de Asientos

Consigna: Se tiene un archivo "pasajeros.txt" con formato nombre;asiento donde asiento puede estar vacio (pasajero sin asiento asignado). Se pide:

  1. Leer el archivo y armar una lista de pasajeros con asiento y otra sin asiento
  2. Detectar colisiones (dos pasajeros con el mismo asiento) y reportarlas
  3. Asignar asientos libres (del 1 al 30) a los pasajeros sin asiento
  4. Grabar el resultado en "asignacion.txt"

Contenido de prueba para pasajeros.txt:

Garcia, Juan;5
Lopez, Maria;
Martinez, Ana;12
Rodriguez, Pedro;5
Fernandez, Luis;
Gomez, Carlos;20
Pistas: Temas: archivos + listas + diccionarios + conjuntos + funciones. Usar un conjunto para asientos ocupados. Detectar colision = asiento ya esta en el conjunto cuando lo queres agregar.
Ver solucion 
def leer_pasajeros(nombre_archivo):
    con_asiento = []
    sin_asiento = []
    colisiones = []
    asientos_usados = set()
    try:
        arch = open(nombre_archivo, "rt")
        for linea in arch:
            linea = linea.rstrip('\n')
            nombre, asiento = linea.split(';')
            if asiento == "":
                sin_asiento.append(nombre)
            else:
                num = int(asiento)
                if num in asientos_usados:
                    colisiones.append((nombre, num))
                else:
                    asientos_usados.add(num)
                    con_asiento.append((nombre, num))
    except FileNotFoundError as msg:
        print("No encontrado:", msg)
    except OSError as msg:
        print("Error:", msg)
    finally:
        try:
            arch.close()
        except NameError:
            pass
    return con_asiento, sin_asiento, colisiones, asientos_usados

def asignar_libres(sin_asiento, asientos_usados, max_asientos=30):
    nuevos = []
    asiento_actual = 1
    for nombre in sin_asiento:
        while asiento_actual in asientos_usados and asiento_actual <= max_asientos:
            asiento_actual += 1
        if asiento_actual <= max_asientos:
            nuevos.append((nombre, asiento_actual))
            asientos_usados.add(asiento_actual)
            asiento_actual += 1
        else:
            print(f"Sin asiento disponible para {nombre}")
    return nuevos

def grabar_asignacion(con_asiento, nuevos, nombre_salida):
    try:
        arch = open(nombre_salida, "wt")
        todos = con_asiento + nuevos
        todos.sort(key=lambda x: x[1])
        for nombre, asiento in todos:
            arch.write(nombre + ';' + str(asiento) + '\n')
    except OSError as msg:
        print("Error:", msg)
    finally:
        try:
            arch.close()
        except NameError:
            pass

# Programa principal
con_asiento, sin_asiento, colisiones, usados = leer_pasajeros("pasajeros.txt")
if colisiones:
    print("COLISIONES detectadas:")
    for nombre, asiento in colisiones:
        print(f"  {nombre} quiere asiento {asiento} (ya ocupado)")
    sin_asiento += [n for n, a in colisiones]
nuevos = asignar_libres(sin_asiento, usados)
grabar_asignacion(con_asiento, nuevos, "asignacion.txt")
print("Asignacion grabada en asignacion.txt")
